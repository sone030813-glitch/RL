# 二、MDP 形式化定义

## 2.1 五元组

MDP = (S, A, P, c, γ)



| 符号 | 含义 | 数学定义 |
|:-----|:-----|:---------|
| S | 状态空间 | 可以是有限、可数或连续 |
| A | 动作空间 | 可以是有限、可数或连续 |
| P | 转移核 | P ∈ P(S | S×A)，给定 (s,a) 下一状态的分布 |
| c | 成本函数 | c ∈ B_b(S×A)，有界可测函数 |
| γ | 折扣因子 | γ ∈ (0,1) |

## 2.2 历史空间

H_n := (S × A)^n × S



表示到时刻 n 的"可容许历史"：(s_0, a_0, s_1, a_1, ..., s_{n-1}, a_{n-1}, s_n)

## 2.3 策略

**确定性策略**：α_n : H_n → A（从历史映射到动作）

**松弛策略（Relaxed）**：π_n : H_n → P(A)（从历史映射到动作分布）

## 2.4 价值函数

**标准形式（τ = 0）**：
V^α(s) = E_s^α [ Σ_{n=0}^∞ γ^n c(s_n, a_n) ]



**带熵正则（τ > 0）**：
V^π_τ(s) = E_s^π [ Σ_{n=0}^∞ γ^n ( ∫_A c(s_n,a) π_n(da) + τ·KL(π_n|μ) ) ]

# 三、相对熵 / KL 散度

## 3.1 绝对连续

若 ν, μ ∈ P(A)，且 μ(B) = 0 ⟹ ν(B) = 0 对所有 B ∈ B(A)，
则称 ν 关于 μ **绝对连续**，记作 ν << μ

## 3.2 KL 散度定义

KL(ν|μ) = { ∫_A ln(dν/dμ) ν(da),  若 ν << μ
{ +∞,                    否则



## 3.3 关键性质

**非负性**：KL(ν|μ) ≥ 0

**等号条件**：KL(ν|μ) = 0 当且仅当 ν = μ

**证明思路**：利用 s·ln(s) ≥ s - 1（对 s ≥ 0），等号当且仅当 s = 1

## 3.4 变分公式（Variational Formula）

对 f ∈ B_b(A)：
inf_{ν∈P(A)} ( ∫_A f dν + KL(ν|μ) ) = -ln ∫_A e^{-f} μ(da)



**最优解**：
dν*/dμ (a) = e^{-f(a)} / ∫_A e^{-f(a')} μ(da')



这就是 **Softmax/Gibbs 分布**！

## 3.5 Donsker-Varadhan 公式（对偶形式）

KL(ν|μ) = sup_{g∈C_b(A)} ( ∫_A g(a) ν(da) - ln ∫_A e^{g(a)} μ(da) )



**性质**：KL(ν|μ) 关于 (ν, μ) 是凸的、下半连续的

# 四、动态规划原理（DPP）

## 4.1 Bellman 方程（τ = 0）

**Theorem (DPP, τ = 0)**：最优价值函数 V* ∈ B_b(S) 是以下方程的唯一解：

V*(s) = min_{a∈A} [ c(s,a) + γ ∫_S V*(s') P(ds'|s,a) ]



**最优策略**：存在 f* : S → A 使得 f*(s) ∈ argmin_a Q*(s,a)

## 4.2 Soft Bellman 方程（τ > 0）

**Theorem (DPP, τ > 0)**：最优价值函数 V*_τ 是以下方程的唯一解：

Vτ(s) = inf{m∈P(A)} ∫_A ( c(s,a) + τ·ln(dm/dμ)(a) + γ ∫_S V_τ(s') P(ds'|s,a) ) m(da)



**最优策略（Softmax 形式）**：
π*(a|s) ∝ exp(-Q*_τ(s,a)/τ) · μ(a)



## 4.3 Q 函数

**定义**：
Q^π(s,a) = c(s,a) + γ ∫∫_{S×A} ( Q^π(s',a') + τ·ln(dπ/dμ)(a'|s') ) π(da'|s') P(ds'|s,a)



**V 和 Q 的关系**：
V^π(s') = ∫_A ( Q^π(s',a') + τ·ln(dπ/dμ)(a'|s') ) π(da'|s')



## 4.4 为什么 Bellman 方程有唯一解？

**Banach 不动点定理**：
若 (M, d) 是完备度量空间，F: M → M 是压缩映射（∃ρ∈(0,1), d(F(x),F(x')) ≤ ρ·d(x,x')），
则存在唯一 x* 使得 F(x*) = x*

**应用**：Bellman 算子 T 是 γ-压缩：
||Tu - Tv||∞ ≤ γ ||u - v||∞

# 五、"Solving RL" 的定义

我们说"解决了 RL 问题"，当且仅当能找到 **近似最优策略**，在以下条件下：

1. **不知道** c 和 P
2. 自选 γ > 0, τ ≥ 0
3. 能访问 **模拟器**，可以反复采样
4. 模拟器从 s ~ ρ ∈ P(S) 初始化，运行到终止或重置

**金融意义**：市场的 P 和 c 不可得，只能通过历史数据/回测/模拟来逼近最优策略

# 六、Q-learning

## 6.1 ε-greedy Q-learning 算法（τ = 0）

输入: 参数化 Q_θ
for n = 0, 1, ..., N_episodes do
清空 memory buffer
for t = 0, 1, ..., N_steps do
观察状态 s_t
动作选择:
以概率 1-ε: a_t = argmin_a Q_θ(s_t|a)
以概率 ε: a_t ~ μ（随机探索）
执行 a_t，得到 c_t, s_{t+1}
存储 (s_t, a_t, c_t, s_{t+1}) 到 memory
end for
从 memory 采样 batch {(s_j, a_j, c_j, s_{j+1})}{j=1}^N
计算 target: v_j = c_j + γ min{a'} Q_{θ_n}(s_{j+1}, a')
损失函数: L(θ) = Σ_j |v_j - Q_θ(s_j, a_j)|²
更新参数: θ_{n+1} = θ_n - η ∇_θ L(θ)
end for



## 6.2 Soft Q-learning（τ > 0）

**DPP 方程**：
Q_τ(s,a) = c(s,a) + γ ∫S inf{m∈P(A)} ( Q_τ(s',a') + τ·ln(dm/dμ)(a') ) m(da') P(ds'|s,a)



**重写为期望形式**：
0 = E_{s'~P(·|s,a), a'~π_τ(·|s')} [ c(s,a) + γ(Q_τ(s',a') + τ·ln(dπ*/dμ)(a'|s')) - Q*_τ(s,a) ]



**TD 更新**：
Q_{k+1}(s,a) = Q_k(s,a) + δ_k [ c(s,a) + γ(Q_k(s',a') + τ·ln(dπ_k/dμ)(a'|s')) - Q_k(s,a) ]



其中：
- s' ~ P(·|s,a)
- π_k(a|s) ∝ exp(-Q_k(s,a)/τ) · μ(a)
- a' ~ π_k(·|s')

# 七、探索策略（Bandit Methods）

## 7.1 Regret 定义

设有 K 个臂，每个臂 k 有期望回报 r_k，最优臂 k* = argmax_k E[r_k]

**Regret**：
R^π(N) = E [ Σ_{n=0}^N ( r_{k*} - Σ_k r_k π_n(k) ) ]



**性质**：
- R^π(N) ≥ 0
- 若 aN ≤ R^π(N) ≤ AN（线性增长），说明没在学习
- 最优 regret 是 O(log N)

## 7.2 三种经典策略

### Explore-then-Exploit
先每个臂试 M 次，然后永远选均值最优的

### ε-greedy
以 1-ε 选当前最优，以 ε 随机探索
不断更新每个臂的均值估计

### UCB（Upper Confidence Bound）
选置信上界最高的臂
置信上界 = 均值估计 + 不确定性奖励（取决于方差和采样次数）

# 八、函数逼近

## 8.1 为什么需要？

若 S 和 A 是有限的 → 表格法（Tabular）
否则需要近似 V: S → R 或 Q: S×A → R

## 8.2 线性逼近

选择基函数/特征 {φ_k : X → R}_{k=1,...,M}，线性独立

f̂(x; θ) = Σ_{k=1}^M θ_k φ_k(x),  θ ∈ R^M



**优点**：稳定、可解释、易于分析
**缺点**：表达能力有限，需要手工设计特征

## 8.3 神经网络逼近

单隐层前馈网络：
f̂(x; θ) = W^(2) φ(W^(1) x + B^(1)) + B^(2)



其中 θ = (W^(1), B^(1), W^(2), B^(2))

**更复杂架构**：Deep FF, RNN, LSTM, Transformer, CNN, ...

**优点**：表达能力强，自动学特征
**缺点**：黑箱，训练不稳定

# 速查：核心公式

【价值函数】
V^π(s) = E [ Σ γ^t c(s_t, a_t) ]

【Bellman (τ=0)】
V*(s) = min_a [ c(s,a) + γ E[V*(s')] ]

【Soft Bellman (τ>0)】
V_τ(s) = inf_m ∫ ( c + τ ln(m/μ) + γ E[V_τ] ) dm

【最优 Soft 策略】
π*(a|s) ∝ exp(-Q*(s,a)/τ) μ(a)

【Q-learning 更新】
Q ← Q + α [ c + γ min Q' - Q ]

【Soft Q-learning 更新】
Q ← Q + α [ c + γ(Q' + τ ln(π/μ)) - Q ]

【KL 散度】
KL(ν|μ) = ∫ ln(dν/dμ) dν ≥ 0

【变分公式】
inf_ν ( ∫f dν + KL(ν|μ) ) = -ln ∫ e^{-f} dμ
